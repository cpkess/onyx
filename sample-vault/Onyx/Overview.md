# Onyx

## Overview
<!--ai
<A brief introductory summary of what Onyx is and its core storage method.>
-->
Onyx is a note-taking system designed to function as a "connected web of notes" [[Welcome]], [[Synthesis - onyx]]. The system stores all information entirely as plain markdown files, ensuring plain-text transparency and data accessibility [[Welcome]], [[Overview]], [[Synthesis - onyx]].

## Key Themes
<!--ai
<An explanation of the core principles driving the Onyx project.>
-->
The development and functionality of Onyx are centered around three key themes:
* **Connectivity:** The system's core functionality is built upon a networked structure of information contained within a [[Knowledge Base]] [[Overview]], [[Synthesis - onyx]].
* **Incremental Development:** The project follows a structured development plan that prioritizes establishing foundational structures before integrating advanced AI layers [[Overview]], [[Roadmap]], [[Synthesis - onyx]].
* **Plain-Text Transparency:** All data within the vault is stored in a standard, accessible format of plain markdown [[Welcome]], [[Synthesis - onyx]].

## Roadmap
<!--ai
<A description of the project's development phases and planned features.>
-->
The Onyx project follows a structured development plan consisting of two primary stages: establishing foundational structures in Phase 1, followed by the integration of AI layers [[Overview]], [[Roadmap]].

Current and planned developments include:
* **Completed Features:** Leveraging headers as sub-page regeneration for deeper AI generation and the addition of a toolbar for formatting [[Roadmap]].
* **Planned UI/Feature Enhancements:** 
    * Implementation of collapsible headers [[Roadmap]].
    * Moving "Weave" into `AI > Tools` and deleting the redundant `AI > Assist` [[Roadmap]].
    * Rendering markdown in `AI > Chat` [[Roadmap]].
    * Webpage importing capabilities [[Roadmap]].
    * Removal of AI from the top bar [[Roadmap]].
    * Implementation of selectable AI text editing (highlighting sections to show changes) [[Roadmap]].
    * Development of a Tauri Auto Updater using GitHub [[Roadmap]].
    * Moving "Change Vault" into settings [[Roadmap]].
